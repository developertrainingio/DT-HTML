Divs and Spans

The next two sections really shine when combined with CSS.

Div

Divs allow you to separate the different parts of our web page when there isn't an existing tag that will do the job (e.g. aside, section, nav, etc.).

Span

Compared to a div, which separates things into logical sections, span tags are used to highlight a special part of a paragraph, similar to a quote, but allowing for a lot more flexibility in what you highlight.

In this course, which focuses on HTML only, span tags don't appear to do anything. That's because this tag is really designed to work hand in hand with CSS and classes.
